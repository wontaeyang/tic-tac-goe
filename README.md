# Tic-Tac-Goe

This repository contains a simple Golang codebase for a game of Tic-Tac-Toe with a twist.
The CLI application takes an argument for grid size and you can have more intense game.


# Building

Use following make command to build the game

```
make build
```

# Running the game

The compiled binary takes an argument of `size` that creates (n x n) grid size Tic-Tac-Toe.

```
./tictacgoe -size=3

Arrow: move cursor, Space: make move, ESC: exit
[ ]|   |
-----------
   |   |
-----------
   |   |
```
